# WebAudioPlayer

一款 **网页音频播放器** ，主要是用来学习 Web Audio API 和 FileSystem API。

## 初始化项目
- [使用 pnpm](/docs/pnpm%E4%BD%BF%E7%94%A8.md)
- 初始化仓库
   ```cmd
   git init
   git add .
   git commit -am "feat: Initial commit 🎉."
   ```
## 配置编码风格
   - 配置 [_.editorconfig_](/.editorconfig)
   - 配置 [_ESLint_](/docs/%E9%85%8D%E7%BD%AEESLint.md)
